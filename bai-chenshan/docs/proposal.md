# 《白衬衫》廉政微电影 · 政府展示全案

> 副标题：白衫易染，初心难守；行稳不偏，方至终点。  
> 类型：廉政题材 · 华语微电影文艺写实 · 约 8′ · 16:9  
> 用途：面向政府 / 纪检宣传单位的项目立意、内容与制作宣发全流程展示

---

## 一、项目立意

《白衬衫》以一件穿旧的白衬衫为视觉母题，讲述干部成长中最危险的一段路：**边界感如何在亲情、同学情与权力感中被一点点松动**。

- **白衬衫**：寒门起跑时的洁净与克制；领口从扣紧到敞开，即是初心从守到失。  
- **风筝线**：远方的线绷成「底线」；断线落空处，是侥幸心理崩塌的瞬间。  
- **警示句**：「白袍点墨，终不可湔。初心易得，始终难守；权力无界，终有尽时。」

本片不以猎奇酒局取胜，而以**日常可感的松动过程**完成廉政警示：办入学、打招呼、收礼表、替人托底——每一步都「说得通」，合在一起却走向不可挽回。

## 二、故事梗概

同学会上，同窗起哄夸赞已是处级干部的陈云峰「实力担当」「一人得道」。靠窗而坐的苏晚起初从容观望，随攀附话术渐生思虑；话题转到「金童玉女」旧情，她才流露无奈与惋惜，由此坠入回忆。

回忆里：青年云峰与苏晚泡图书馆、考公苦读；二叔资助人情桌；林荫道与录取通知的誓约；阳台洗净的白衬衫与入职证照——**风筝线仍绷着**。

回神后仍在夜色同学会饭局。随后现实加速：橱窗腕表的迟疑、家宴以茶代酒的坚守、二叔灌酒送礼表、家中决裂手镯碎地、商务旋转包厢的围猎。醉后梦中搓洗不掉领口污渍，敲门声切断一切——纪检人员亮证带走。

终章狱中叠化少年白衬衫与风筝，字幕落定两行点题。

## 六、宣传方式（政府场景）

### 6.1 传播目标

- **对内警示教育**：党员干部、年轻干部培训班、廉政主题党日。  
- **对外价值宣示**：文艺精品呈现「执纪必严」与「守住初心」的双重叙事。  
- **可转述金句**：白衫易染两句 +「办一件小事的侥幸」。

### 6.2 渠道矩阵

| 层级 | 渠道 | 动作 |
|------|------|------|
| 主阵地 | 机关内网 / 学习专栏 / 纪委宣传口 | 正片 + 讲解词 + 观后讨论提纲 |
| 培训场 | 干部学院 / 专题党课 | 切段放映（同学会攀附 / 收礼表 / 决裂 / 敲门）+ 研讨 |
| 新媒体 | 政务号短视频（合规剪辑） | 30–60″ 金句短片、白衬衫母题 KV |
| 线下展映 | 廉政教育基地、专题展厅 | 本预览站 + 定妆图册 + 正片循环 |

### 6.3 宣发节奏建议

1. **立项预热**：立意短片 / KV（白衬衫+风筝线）· 定向机关领导审阅。  
2. **制作中透**：资产冻结与制作链路展厅（本站）· 增强过程信任。  
3. **成片首映**：机关礼堂 / 培训班首场 · 领导干部带看。  
4. **常态化**：党课模块化切片 + 观后问卷 · 形成可复制课时包。

### 6.4 观后研讨切入（示例）

- 「打个招呼」何时越过边界？  
- 亲情资助与利益输送如何辨识？  
- 苏晚的劝诫为何失败、为何仍必要？  
- 白衬衫洗净不掉的梦境对应何种心境？
