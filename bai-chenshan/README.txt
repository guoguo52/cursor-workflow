# 《白衬衫》政府展示全案站

项目路径：`_projects/bai-chenshan/05_preview_site/`  
**本地端口：8770**（勿占用乐动敦煌 8765）

## 打开方式

1. 双击 `start_local_server.bat`，浏览器打开 http://127.0.0.1:8770/  
2. 或：

```bash
cd "_projects/bai-chenshan/05_preview_site"
python -m http.server 8770
```

也可双击 `START_HERE.html` / `index.html`（图册相对路径在 file:// 下可用；附件链建议用本地服务以便审阅父级 md/txt）。
